## Overview

The **Favourites** page gives quick access to items you use frequently. Any item you mark as a favourite is listed here, regardless of the folder it belongs to.

> 🔔 The favourites feature must be enabled by your administrator (**Settings → Items → Enable favourites**). If the Favourites menu entry is not visible, the feature is disabled on your instance.

> 🔔 Favourites are personal — only you can see your list.

> 🔔 The Favourites page is not available for administrator accounts.

---

## Marking an item as a favourite

From the item list in any folder:

1. Hover over the item row.
2. Click the **star icon** (☆) in the item's action bar.
3. The star turns solid (★) and the item is added to your favourites.

To remove a favourite, click the star icon again. You can also remove it directly from the Favourites page.

---

## Favourites page

The Favourites page shows all your marked items in a table with the following columns:

| Column | Content |
|--------|---------|
| **Actions** | Open URL (external link icon) · Remove from favourites (trash icon) |
| **Label** | Item name — click to navigate to the item in its folder |
| **Description** | Item description |
| **Group** | Folder the item belongs to |

If you have no favourites yet, an informational message is shown instead of the table.

---

## Highlighting favourites in the item list

When the **Highlight favourite items** option is enabled by the administrator, favourite items are visually distinguished in the main item list (e.g., with a coloured background or badge), so you can identify them without navigating to the Favourites page.
